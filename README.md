# dom-api-proxy
- creating native dom apis using htmlparser2, css-select and es2015 proxies
- using this we will be able to use native dom apis is nodejs environment
- to see the project in action head on to [repl.it](https://repl.it/@kevjose91/proxy-dom-api#index.js) and press `run`, this will fire up the node engine and extract the selector values logged (`console.log`) in the `main` function
